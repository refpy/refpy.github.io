Limit States#

Library of DNV limit states

refpy.limit_states.temperature_derating_stress(material: ndarray, temperature: ndarray) → ndarray[source]#

Calculate the temperature derating stress of a material.

Parameters:

  • material (numpy.ndarray) – The types of materials (e.g., ‘CMn’, ‘13CR’, ‘22Cr’, ‘25CR’).

  • temperature (numpy.ndarray) – The temperatures at which to calculate the derating stress.

Returns:

The derating stress values for the given materials and temperatures.

Return type:

numpy.ndarray

Raises:

ValueError – If a material is not supported.
