Limit States#

Library of limit states

refpy.limit_states.lcc(od: float, wt: float, ca: float, linepipe_type: str, temp: ndarray, pi: ndarray, pe: ndarray, eaf: ndarray, bm: ndarray, safety_class: str) → ndarray[source]#

Calculate the limit state function for local buckling.

Parameters:

  • od (float) – Outer diameter of the pipe.

  • wt (float) – Wall thickness of the pipe.

  • ca (float) – Corrosion allowance.

  • linepipe_type (str) – Type of the linepipe (‘smls’ or ‘sawl’).

  • temp (np.ndarray) – Temperature array.

  • pi (np.ndarray) – Internal pressure array.

  • pe (np.ndarray) – External pressure array.

  • eaf (np.ndarray) – Effective axial force array.

  • bm (np.ndarray) – Bending moment array.

  • safety_class (str) – Safety class (‘Low’, ‘Medium’, or ‘High’).

Returns:

lcc – Limit state function for local buckling.

Return type:

np.ndarray
