Pipe Properties#

Library of pipeline main properties

class refpy.pipe_properties.Pipe(*, outer_diameter=0.0, wall_thickness=0.0, coating_thickness=0.0, corrosion_allowance=0.0, youngs_modulus=0.0)[source]#

Bases: object

Class representing a pipeline section with geometric and material properties. Supports both scalar and array inputs for calculations.
